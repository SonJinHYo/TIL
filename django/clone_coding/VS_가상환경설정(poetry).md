VS에서 Ctrl+백틱 으로 터미널을 쉽게 킬 수 있다.

## 가상환경 설정

터미널에 다음과 같이 입력한다.

1. `poetry init`
   - 프로젝트 초기 세팅을 한다.
   - 패키지 이름, 버전, 작성자 등등의 이름을 묻는다.
   - 어차피 나중에 수정 가능하니 엔터연타를 해준다.
2. `poetry add django`
   - 가상환경에 django를 설치한다.
   - `poetry add (패키지명)` 으로 가상환경에 패키지 다운이 가능하다
3. `poetry shell`
   - 가상환경 구동
   - 나올땐 `exit`
4. gitignore 생성. 보통 IDE 확장팩에 들어가있다.
